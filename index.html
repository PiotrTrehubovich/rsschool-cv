Code example

Task: To square every digit of a number and concatenate them.

Example: squareDigits(92); // Should return 814 

 
                
                  function squareDigits(num){
                    return Number(num
                        .toString()
                        .split("")
                        .map((i) => Math.pow(i, 2))
                        .join(""));
                  }
